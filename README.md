# Kinto Box

Kinto Box is a ruby client for [Kinto](kinto.readthedocs.io)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'kinto_box'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kinto_box
